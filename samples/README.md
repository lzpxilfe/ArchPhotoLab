# 샘플 프로젝트 템플릿

ArchPhotoLab 실행 흐름(사진/도면 로딩 → 정렬점 편집 → 정합 → 보정 → 내보내기)을 빠르게 검증할 때
`project_templates`의 JSON을 복제해 사용하세요.

## 사용 방법

1. `sample-*.json`을 열어 `photo_path`, `plan_path`를 현재 작업 폴더 기준 실제 파일 경로로 바꿉니다.
2. `결과 저장` 또는 `프로젝트 불러오기`를 통해 상태를 불러옵니다.
3. `정렬`/`품질` 패널에서 오차와 이상점을 확인합니다.
4. 필요 시 점을 수정하고 재정렬해 저장/내보내기까지 반복합니다.

## 템플릿

- `sample-good.json` : 기본 정합 흐름 점검용
- `sample-shadow.json` : 강한 그림자/명암 편차 보정 우선 점검
- `sample-outlier.json` : 일부 대응점이 어긋나는 경우 이상점 경고 확인
