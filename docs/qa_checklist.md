# QA 체크리스트 (v0.1.0)

## 기본 기능

- [ ] 1장 사진 + 1장 도면 로드 후 패널에 파일명/해상도 표시
- [ ] 정합 점을 양쪽 각 4개 이상 추가 및 이동/삭제 가능
- [ ] `자동 정합` 실행 후 overlay 미리보기 동작
- [ ] 정합 품질(평균/중앙값/최대/등급/이상점 개수) 표시
- [ ] 정합이 실패하는 입력(점 부족/불일치)에서 사용자 메시지 노출

## 정합 모드

- [ ] `homography` 정합 동작
- [ ] `affine` 정합 동작
- [ ] `similarity` 정합 동작
- [ ] 점 삭제/재배치 후 재정렬 정합 성공

## 플랫 보정

- [ ] `기록용`, `강한음영완화`, `부드러운정리` 프리셋 변경
- [ ] 강도 슬라이더로 평탄화 정도 변경 반영
- [ ] `원본/평탄화 보기`, `분할 비교` 토글 동작

## 내보내기/복원

- [ ] 3종 PNG(`overlay`, `flat`, `warped`) 기본명 생성
- [ ] 프로젝트 저장/불러오기 후 점, 정합 모드, 품질 상태가 재현되는지 확인
- [ ] 오래된 `format_version` 프로젝트 로드 시 경고/누락 채우기
